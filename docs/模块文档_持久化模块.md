# 💾 持久化模块文档

---

## 一、模块定位与职责

**模块名称**: Storage (持久化模块)

**职责**:
- localStorage 数据存储与读取
- 单词熟练度数据持久化
- 不熟单词列表管理
- 数据加载与初始化

**文件位置**: [js/storage.js](js/storage.js)

---

## 二、存储键定义

```javascript
const STORAGE_KEYS = {
    WORD_PROGRESS: 'adhd-word-progress',
    UNFAMILIAR_WORDS: 'adhd-unfamiliar-words'
};
```

| 存储键 | 存储内容 | 更新时机 |
|--------|----------|----------|
| `adhd-word-progress` | 所有单词的熟练度数据 | 每次熟练度更新后 |
| `adhd-unfamiliar-words` | 不熟单词列表 | 每次添加/移除不熟单词时 |

## 三、核心函数

### 3.1 进度保存与加载

**保存进度**:
```javascript
export function saveProgress() {
    localStorage.setItem(STORAGE_KEYS.WORD_PROGRESS, JSON.stringify(wordDB));
}
```

**加载进度**:
```javascript
export function loadProgress() {
    const saved = localStorage.getItem(STORAGE_KEYS.WORD_PROGRESS);
    if (saved) {
        const savedDB = JSON.parse(saved);
        savedDB.forEach(savedWord => {
            const word = wordDB.find(w => w.id === savedWord.id);
            if (word) {
                Object.assign(word, savedWord);
            }
        });
    }
}
```

**设计要点**:
- 使用 `Object.assign` 合并保存的数据，保留原始对象引用
- 只更新已存在的单词，不添加新单词
- 完整保存整个词库数据（包括熟练度、出现次数等）

---

### 3.2 不熟单词管理

**保存不熟单词**:
```javascript
export function saveUnfamiliarWords() {
    localStorage.setItem(STORAGE_KEYS.UNFAMILIAR_WORDS, JSON.stringify(state.unfamiliarWords));
}
```

**加载不熟单词**:
```javascript
export function loadUnfamiliarWords() {
    const saved = localStorage.getItem(STORAGE_KEYS.UNFAMILIAR_WORDS);
    if (saved) {
        state.unfamiliarWords = JSON.parse(saved);
    }
}
```

**添加不熟单词**:
```javascript
export function addToUnfamiliar(word) {
    if (!state.unfamiliarWords.some(w => w.id === word.id)) {
        state.unfamiliarWords.push({ ...word });
        saveUnfamiliarWords();
    }
}
```

**移除不熟单词**:
```javascript
export function removeFromUnfamiliar(wordId) {
    state.unfamiliarWords = state.unfamiliarWords.filter(w => w.id !== wordId);
    saveUnfamiliarWords();
    renderUnfamiliarLibrary();
}
```

**设计要点**:
- 使用深拷贝 `{ ...word }` 避免引用问题
- 添加前检查是否已存在，避免重复
- 移除后自动重新渲染不熟词库UI

## 六、初始化流程

```
1. 页面加载完成 (DOMContentLoaded)
2. loadProgress() 加载单词熟练度数据
3. loadUnfamiliarWords() 加载不熟单词列表
4. 绑定键盘事件监听器
5. 系统准备就绪
```
