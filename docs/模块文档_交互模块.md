## 二、核心函数详解

### 2.1 键盘事件处理 - `handleKeyDown(e)`

**功能**: 全局键盘事件入口，负责分发所有键盘操作

**核心代码**:
```javascript
export function handleKeyDown(e) {
    if (e.target.tagName === 'INPUT' || e.target.tagName === 'TEXTAREA') return;
    
    switch (e.code) {
        case 'Space':
            e.preventDefault();
            if (state.currentState === STATE.SHOW_ANSWER) {
                transition('NEXT_WORD');
            } else {
                transition('TOGGLE_PLAY');
            }
            break;
        // ...其他按键处理
    }
}
```

**设计要点**:
- **输入框保护**: 如果焦点在输入框中，不处理快捷键
- **Space键特殊处理**: 根据当前状态决定是播放/暂停还是继续下一个单词
- **事件分发**: 根据按键类型分发到不同的处理逻辑

---

### 2.2 快捷键映射表

#### 核心操作区
| 按键 | 事件 | 功能描述 | 支持状态 |
|------|------|----------|----------|
| **Space** | `TOGGLE_PLAY` / `NEXT_WORD` | 播放/暂停 / 继续下一个 | 全局 |
| **K** | `SHOW_MEANING` / `KNOW` | 显示释义 / 确认知道 | PLAYING / WAIT_RESPONSE / SHOW_ANSWER |
| **J** | `UNFAMILIAR` | 标记不熟 | PLAYING / WAIT_RESPONSE |
| **A** | `goPrevWord()` | 历史后退 | 全局 |
| **D** | `goNextWord()` | 历史前进 | 全局 |

#### 系统控制区
| 按键 | 功能描述 | 支持状态 |
|------|----------|----------|
| **1/2/3/5** | 切换速度档位 | 全局 |
| **M** | 切换训练模式 | 全局 |
| **L** | 打开完整词库 | 全局 |
| **W** | 切换历史面板 | 全局 |
| **U** | 打开不熟词库 | 全局 |

**快捷键优先级**:
1. 输入框保护（最高）
2. 全局快捷键（L/M/W/U/A/D/数字键）
3. 训练控制键（Space/K/J）

### 2.3 速度档位调节

**速度配置**:
| 档位 | 语音速率 | 间隔时间 | 适用场景 |
|------|----------|----------|----------|
| 1x | 0.8x | 2000ms | 热身/慢速学习 |
| 2x | 1.5x | 1500ms | 正常训练 |
| 3x | 2.2x | 1000ms | 快速反应 |
| 5x | 3.0x | 500ms | 极限训练 |

**核心代码**:
```javascript
case 'Digit1':
case 'Digit2':
case 'Digit3':
case 'Digit5':
    const speed = parseInt(e.key);
    if ([1, 2, 3, 5].includes(speed)) {
        state.currentSpeed = speed;
        updateModeIndicator();
    }
    break;
```

### 2.5 词库弹窗管理

#### 完整词库

**打开词库**:
```javascript
export function openWordLibrary() {
    const modal = document.getElementById('word-library-modal');
    if (modal) {
        modal.style.display = 'block';
        updateLibraryStats();
        renderWordCards(wordDB);
    }
}
```

**关闭词库**:
```javascript
export function closeWordLibrary() {
    const modal = document.getElementById('word-library-modal');
    if (modal) {
        modal.style.display = 'none';
```

### 2.6 词库过滤功能

**功能**: 根据分类、主题、难度等级过滤单词

**过滤类型**:
| 过滤类型 | 参数值 | 过滤字段 |
|----------|--------|----------|
| 分类 | `noun`, `verb`, `adjective`, `adverb`, `preposition` | `category` |
| 主题 | `academic`, `daily`, `technology`, `economy`, `society`, `emotion` | `topic` |
| 难度 | `basic`, `intermediate`, `advanced` | `level` |

**核心代码**:
```javascript
export function filterWords(filterType) {
    document.querySelectorAll('.filter-btn').forEach(btn => btn.classList.remove('active'));
    event.target.classList.add('active');
    
    let filteredWords = wordDB;
    
    switch (filterType) {
        case 'noun':
            filteredWords = wordDB.filter(w => w.category === '名词');
            break;
        case 'verb':
            filteredWords = wordDB.filter(w => w.category === '动词');
            break;
        // ...其他过滤逻辑
    }
    
    renderWordCards(filteredWords);
}
```

## 五、事件处理流程

### 典型流程: 用户按K键表示"知道"

```
1. 用户按下 K 键
2. handleKeyDown(e) 捕获事件
3. 判断当前状态:
   - PLAYING → transition('SHOW_MEANING')
   - WAIT_RESPONSE → transition('KNOW')
   - SHOW_ANSWER → transition('KNOW')
4. transition() 执行对应逻辑
5. 播放音效、更新熟练度、显示下一个单词
```

### 典型流程: 用户按L键打开词库

```
1. 用户按下 L 键
2. handleKeyDown(e) 捕获事件
3. 调用 openWordLibrary()
4. 显示词库弹窗
5. 调用 updateLibraryStats() 更新统计
6. 调用 renderWordCards(wordDB) 渲染卡片
```
