## 一、模块定位与职责

**模块名称**: CSS Styles (样式模块)

**职责**:
- 颜色主题定义
- 布局组件样式
- 动画效果实现
- 响应式设计
- 视觉反馈效果

**文件位置**: [css/main.css](css/main.css)

## 二、颜色主题

### 2.1 CSS变量定义

```css
:root {
    --bg-color: #0f0f11;        /* 深色背景 */
    --text-main: #ffffff;       /* 主文字 */
    --text-dim: #555555;        /* 暗淡文字 */
    --accent-green: #39ff14;    /* 知道/成功 */
    --accent-red: #ff073a;      /* 不熟/失败 */
    --accent-yellow: #fefe33;   /* 警告/释义 */
    --accent-blue: #00d4ff;     /* 词库/分类 */
    --accent-purple: #b347ff;   /* 主题标签 */
    --accent-orange: #ff9500;   /* 等级标签 */
    --accent-pink: #ff2d55;     /* 装饰色 */
}
```

### 2.2 颜色用途

| 颜色变量 | 用途 | 示例组件 |
|----------|------|----------|
| `--bg-color` | 全局背景 | `body`, `#main-stage` |
| `--text-main` | 主要文字 | `#word-display`, `#meaning-display` |
| `--text-dim` | 次要文字 | `#phonetic-display`, 标签 |
| `--accent-green` | 成功状态 | Combo显示、知道按钮 |
| `--accent-red` | 失败状态 | 不熟指示器、不熟词库 |
| `--accent-yellow` | 警告/重点 | 状态指示器、释义 |
| `--accent-blue` | 中性操作 | 词库弹窗、分类标签 |
| `--accent-purple` | 主题标签 | 主题分类标签 |
| `--accent-orange` | 等级标签 | 难度等级标签 |

## 三、布局组件

### 3.1 全局重置

```css
body, html { 
    margin: 0; 
    padding: 0; 
    width: 100%; 
    height: 100%; 
    background-color: var(--bg-color); 
    color: var(--text-main); 
    font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif; 
    overflow: hidden; 
    display: flex; 
    flex-direction: column; 
    user-select: none; 
}
```

**设计要点**:
- 使用 flexbox 布局
- 禁止文字选中
- 隐藏滚动条

### 3.2 顶部栏 (#header)

```css
#header { 
    display: flex; 
    justify-content: space-between; 
    align-items: center; 
    padding: 20px 40px; 
    font-weight: bold; 
    text-transform: uppercase; 
    letter-spacing: 2px; 
}
```

**包含元素**:
- `#mode-speed-indicator` - 模式和速度指示
- `#combo-container` - Combo显示
- `#status-indicator` - 状态指示

### 3.3 主训练区 (#main-stage)

```css
#main-stage { 
    flex: 1; 
    display: flex; 
    flex-direction: column; 
    justify-content: center; 
    align-items: center; 
    text-align: center; 
    padding: 0 20px; 
}
```

**包含元素**:
- `#start-hint` - 开始提示
- `#word-display` - 单词显示
- `#phonetic-display` - 音标显示
- `#meaning-display` - 释义显示
- `#example-display` - 例句显示

### 3.4 底部栏 (#footer)

```css
#footer { 
    padding: 20px; 
    display: flex; 
    justify-content: space-between; 
    font-size: 0.9rem; 
    color: var(--text-dim); 
    border-top: 1px solid #222; 
}
```

**包含元素**:
- 左侧：核心操作快捷键提示
- 中间：`#stats-board` 统计面板
- 右侧：系统控制快捷键提示

## 七、动画效果

```css
#unfamiliar-indicator { 
    position: fixed; 
    top: 50%; 
    left: 50%; 
    transform: translate(-50%, -50%); 
    font-size: 8rem; 
    font-weight: 900; 
    color: var(--accent-red); 
    opacity: 0; 
    pointer-events: none; 
    z-index: 999; 
    text-shadow: 0 0 30px var(--accent-red); 
}

.show-unfamiliar { animation: unfamiliar-pulse 0.5s ease-out; }
@keyframes unfamiliar-pulse {
    0% { opacity: 0; transform: translate(-50%, -50%) scale(0.5); }
```

## 十、响应式设计

### 10.1 字体大小适配

当前设计使用固定 `rem` 单位，建议在实际生产环境中添加媒体查询：

```css
@media (max-width: 768px) {
    #word-display { font-size: 3rem; }
    #meaning-display { font-size: 1.5rem; }
    #header { padding: 10px 20px; }
    #footer { padding: 10px; font-size: 0.75rem; }
}
```
