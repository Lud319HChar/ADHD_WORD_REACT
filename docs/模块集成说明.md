## 三、初始化流程

### 3.1 页面加载流程

```
1. 浏览器加载 index.html
2. 加载并解析 main.css
3. 执行 <script type="module"> 中的代码
4. 导入所有模块
5. DOMContentLoaded 事件触发
6. loadProgress() 加载单词熟练度
7. loadUnfamiliarWords() 加载不熟单词
8. 绑定键盘事件监听器
9. 系统进入 IDLE 状态
```

### 3.2 模块初始化顺序

```
1. constants.js   → 常量定义（无依赖）
2. wordDB.js      → 词库数据（依赖 constants.js）
3. state.js       → 状态管理（依赖 constants.js）
4. audio.js       → 音频模块（依赖 state.js, constants.js）
5. storage.js     → 持久化模块（依赖 state.js, wordDB.js）
6. core.js        → 核心引擎（依赖所有模块）
7. interaction.js → 交互模块（依赖核心模块）
8. index.html     → 页面初始化（依赖交互模块）
```

## 四、核心交互流程

### 4.1 训练开始流程

```
用户按 Space
    ↓
handleKeyDown('Space')
    ↓
transition('TOGGLE_PLAY')
    ↓
invalidateSession()
    ↓
updateState(PLAYING)
    ↓
playNextWord()
    ↓
getRandomWord()
    ↓
displayWord(word)
    ↓
speakWord(word.word)
    ↓
safeScheduleTimer(transition('SHOW_MEANING'), gap)
```

### 4.2 用户反馈流程

```
用户按 K (知道)
    ↓
handleKeyDown('KeyK')
    ↓
transition('KNOW')
    ↓
playKnowSound()
    ↓
updateCombo(combo + 1)
    ↓
updateMastery(word, true)
    ↓
saveProgress()
    ↓
playNextWord()
```

```
用户按 J (不熟)
    ↓
handleKeyDown('KeyJ')
    ↓
transition('UNFAMILIAR')
    ↓
showAnswer()
    ↓
showUnfamiliarIndicator()
    ↓
playUnfamiliarSound()
    ↓
updateCombo(0)
    ↓
updateMastery(word, false)
    ↓
addToUnfamiliar(word)
    ↓
saveProgress()
```

## 五、模块间通信

### 5.1 状态共享

所有模块通过 `state.js` 共享全局状态：

```javascript
// 状态对象结构
state = {
    currentState: 'IDLE',
    currentWord: null,
    historyStack: [],
    combo: 0,
    // ...
}
```

### 5.2 事件驱动

交互模块 → 核心引擎：通过 `transition(event)` 触发状态转移

```javascript
// interaction.js
transition('KNOW');

// core.js
export function transition(event) {
    // 处理状态转移
}
```

### 5.3 函数调用链

```
handleKeyDown()
    └── transition()
            ├── playKnowSound() / playUnfamiliarSound()
            ├── updateCombo()
            ├── updateMastery()
            │       └── saveProgress()
            ├── addToUnfamiliar()
            │       └── saveUnfamiliarWords()
            └── playNextWord()
                    ├── getRandomWord()
                    ├── displayWord()
                    │       └── speakWord()
                    └── safeScheduleTimer()
```

## 八、扩展建议

### 8.1 新增训练模式

1. 在 `constants.js` 中添加新的模式常量
2. 在 `core.js` 中添加模式切换逻辑
3. 在 `interaction.js` 中添加模式切换的UI更新
4. 在 `main.css` 中添加模式相关样式

### 8.2 新增数据统计

1. 在 `state.js` 中添加统计字段
2. 在 `core.js` 中更新统计逻辑
3. 在 `storage.js` 中保存统计数据
4. 在 `index.html` 中添加统计展示UI

### 8.3 新增音效

1. 在 `constants.js` 中添加音效配置
2. 在 `audio.js` 中添加音效播放函数
3. 在 `core.js` 中调用新音效
