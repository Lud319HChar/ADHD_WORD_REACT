## 二、核心函数详解

### 2.1 状态机核心 - `transition(event)`

**功能**: 处理所有状态转移请求，是整个系统的核心控制函数

**参数**:
| 参数 | 类型 | 说明 |
|------|------|------|
| `event` | String | 触发的事件类型 |

**支持的事件类型**:
| 事件 | 触发条件 | 行为描述 |
|------|----------|----------|
| `TOGGLE_PLAY` | Space键 | 播放/暂停切换 |
| `SHOW_MEANING` | 定时/手动 | 显示单词释义 |
| `KNOW` | K键 | 用户确认知道该单词 |
| `UNFAMILIAR` | J键 | 用户标记不熟 |
| `NEXT_WORD` | Space(答案展示后) | 进入下一个单词 |
| `SWITCH_MODE` | M键 | 切换训练模式 |

**核心代码**:
```javascript
export function transition(event) {
    if (state.isProcessingInput && event !== 'TOGGLE_PLAY') return;
    
    const nextState = getNextState(state.currentState, event);
    
    switch (event) {
        case 'TOGGLE_PLAY':
            if (state.currentState === STATE.IDLE) {
                invalidateSession();
                updateState(STATE.PLAYING);
                playNextWord();
            } else {
                invalidateSession();
                updateState(STATE.IDLE);
                // 隐藏单词显示...
            }
            break;
        // ...其他事件处理
    }
}
```

**状态转移流程图**:
```
IDLE ───(Space)──→ PLAYING ───(定时/K)──→ WAIT_RESPONSE
 ↑                      │                       │
 │              (J)     │                       │
 │                      ↓                       │
 │               SHOW_ANSWER ←──────────────────┘
 │                      │
 │                      ↓ (Space/K)
 └───────────────── NEXT_WORD
```

### 2.2 动态加权随机选词 - `getRandomWord()`

**功能**: 基于熟练度和遗忘曲线动态计算权重，实现智能复习

**算法流程**:
1. **筛选可用单词**: 只选择 `nextReviewTime <= 当前时间` 的单词
2. **计算权重**:
   - `decay = 0.95^((当前时间 - lastSeenTimeStamp) / 3600000)` - 时间衰减因子
   - `weight = (1 - masteryScore/100) * decay + 随机扰动(0.3)`
3. **加权随机选择**: 根据权重概率分布选择单词

**核心代码**:
```javascript
export function getRandomWord() {
    const now = Date.now();
    const availableWords = wordDB.filter(w => now >= w.nextReviewTime);
    
    if (availableWords.length === 0) {
        return wordDB[Math.floor(Math.random() * wordDB.length)];
    }
    
    const weights = availableWords.map(w => {
        const decay = Math.pow(0.95, (now - w.lastSeenTimeStamp) / 3600000);
        return (1 - w.masteryScore / 100) * decay + Math.random() * 0.3;
    });
    
    const totalWeight = weights.reduce((a, b) => a + b, 0);
    let random = Math.random() * totalWeight;
    
    for (let i = 0; i < availableWords.length; i++) {
        random -= weights[i];
        if (random <= 0) return availableWords[i];
    }
    
    return availableWords[availableWords.length - 1];
}
```

**设计原理**:
- **熟练度越低，权重越高**：`(1 - masteryScore/100)` 确保不熟悉的单词更容易被选中
- **时间衰减**：越久没复习的单词权重越高
- **随机扰动**：增加随机性，避免机械重复

### 2.3 单词显示 - `displayWord(word)`

**功能**: 在主训练区渲染单词信息，并更新历史栈

**处理流程**:
1. 显示单词、音标、释义容器
2. 调用语音合成朗读单词
3. 更新今日训练计数
4. 将单词加入历史栈
5. 渲染历史面板

**核心代码**:
```javascript
export function displayWord(word) {
    startHint.style.display = 'none';
    wordDisplay.style.display = 'block';
    phoneticDisplay.style.display = 'block';
    
    wordDisplay.textContent = word.word;
    phoneticDisplay.textContent = word.phonetic;
    meaningDisplay.textContent = word.meaning;
    exampleDisplay.textContent = word.example;
    
    meaningDisplay.classList.remove('show-meaning');
    exampleDisplay.classList.remove('show-meaning');
    
    speakWord(word.word);
    
    state.currentWord = word;
    state.todayCount++;
    updateStats();
    
    if (state.historyPointer === state.historyStack.length - 1) {
        state.historyStack.push({ ...word });
    } else {
        state.historyStack = state.historyStack.slice(0, state.historyPointer + 1);
        state.historyStack.push({ ...word });
    }
    state.historyPointer = state.historyStack.length - 1;
    
    renderHistoryPanel();
}
```

### 2.4 熟练度更新 - `updateMastery(word, success)`

**功能**: 根据用户反馈更新单词熟练度，并计算下次复习时间

**参数**:
| 参数 | 类型 | 说明 |
|------|------|------|
| `word` | Object | 单词对象 |
| `success` | Boolean | 是否知道该单词 |

**更新规则**:
| 操作 | masteryScore变化 | nextReviewTime计算 |
|------|------------------|-------------------|
| 知道 (K键) | +10 (上限100) | `当前时间 + 1.5^(masteryScore/20) * 60秒` |
| 不熟 (J键) | -15 (下限0) | `当前时间` (立即进入复习队列) |

**核心代码**:
```javascript
export function updateMastery(word, success) {
    if (success) {
        word.masteryScore = Math.min(100, word.masteryScore + 10);
        const interval = Math.pow(1.5, Math.floor(word.masteryScore / 20));
        word.nextReviewTime = Date.now() + interval * 60000;
    } else {
        word.masteryScore = Math.max(0, word.masteryScore - 15);
        word.nextReviewTime = Date.now();
    }
    word.appearCount++;
    word.lastSeenTimeStamp = Date.now();
    saveProgress();
}
```

**记忆曲线设计**:
- 熟练度越高，复习间隔呈指数增长
- 熟练度低于60时，间隔增长较慢（需要更频繁复习）
- 熟练度高于80时，间隔显著增长（已牢固掌握）

### 2.5 连击系统 - `updateCombo(newCombo)`

**功能**: 更新连击计数并触发视觉效果

**Combo等级**:
| 阈值 | 样式类 | 视觉效果 |
|------|--------|----------|
| 0-4 | `combo-low` | 灰色，正常大小 |
| 5-9 | `combo-mid` | 黄色，放大1.1倍 |
| 10-19 | `combo-high` | 橙色发光，放大1.2倍 |
| ≥20 | `combo-god` | 绿色发光，放大1.3倍 + 全屏边框 |

**核心代码**:
```javascript
export function updateCombo(newCombo) {
    state.combo = newCombo;
    comboContainer.textContent = `COMBO ${state.combo}`;
    
    if (state.combo >= 20) comboContainer.className = 'combo-god';
    else if (state.combo >= 10) comboContainer.className = 'combo-high';
    else if (state.combo >= 5) comboContainer.className = 'combo-mid';
    else comboContainer.className = 'combo-low';
    
    if (state.combo >= 20 && !document.body.classList.contains('god-mode-border')) {
        document.body.classList.add('god-mode-border');
    } else if (state.combo < 20) {
        document.body.classList.remove('god-mode-border');
    }
}
```

### 2.6 历史漫游功能

#### `goPrevWord()` - 上一个单词
```javascript
export function goPrevWord() {
    if (state.historyPointer > 0) {
        state.historyPointer--;
        const word = state.historyStack[state.historyPointer];
        displayWordFromHistory(word);
    }
}
```

#### `goNextWord()` - 下一个单词
```javascript
export function goNextWord() {
    if (state.historyPointer < state.historyStack.length - 1) {
        state.historyPointer++;
        const word = state.historyStack[state.historyPointer];
        displayWordFromHistory(word);
    }
}
```

## 四、关键设计模式

### 4.1 有限状态机 (FSM)
- 严格控制状态转移，避免非法状态
- 防止异步竞态问题（如快速按键）

### 4.2 会话锁机制
- `playSessionID` 单调递增
- 异步操作（语音合成）检查会话ID是否有效
- 无效会话自动熔断

### 4.3 定时器托管
- 所有定时器统一管理在 `activeTimersPool`
- 状态切换时统一清除，防止定时器泄漏

## 五、典型使用场景

### 场景1: 用户开始训练
```
1. 用户按 Space → handleKeyDown → transition('TOGGLE_PLAY')
2. 状态从 IDLE → PLAYING
3. playNextWord() → getRandomWord() → displayWord()
4. 语音合成朗读单词
5. 定时器触发 → transition('SHOW_MEANING')
6. 状态 → WAIT_RESPONSE，显示释义
```

### 场景2: 用户标记"知道"
```
1. 用户按 K → handleKeyDown → transition('KNOW')
2. 播放知道音效 → updateCombo(combo+1)
3. updateMastery(currentWord, true) → 熟练度+10
4. updateState(PLAYING) → playNextWord()
```

### 场景3: 用户标记"不熟"
```
1. 用户按 J → handleKeyDown → transition('UNFAMILIAR')
2. 显示释义 → 播放不熟音效 → updateCombo(0)
3. updateMastery(currentWord, false) → 熟练度-15
4. addToUnfamiliar(currentWord) → 加入不熟词库
5. 状态 → SHOW_ANSWER
```
