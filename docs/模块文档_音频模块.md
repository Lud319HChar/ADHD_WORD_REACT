## 二、核心功能

### 2.1 Web Audio API 音效

#### 音效配置

| 音效类型 | 频率 | 时长 | 波形 | 用途 |
|----------|------|------|------|------|
| KNOW | 880Hz | 0.15s | sine | 用户按K键表示"知道" |
| UNFAMILIAR | 220Hz | 0.3s | sawtooth | 用户按J键表示"不熟" |
| SHOW_MEANING | 440Hz | 0.1s | sine | 显示释义时 |

#### 基础音效播放函数

```javascript
export function playAudio(frequency, duration, type = 'sine') {
    initAudioContext();
    
    const oscillator = audioContext.createOscillator();
    const gainNode = audioContext.createGain();
    
    oscillator.connect(gainNode);
    gainNode.connect(audioContext.destination);
    
    oscillator.frequency.value = frequency;
    oscillator.type = type;
    
    gainNode.gain.setValueAtTime(0.3, audioContext.currentTime);
    gainNode.gain.exponentialRampToValueAtTime(0.01, audioContext.currentTime + duration);
    
    oscillator.start(audioContext.currentTime);
    oscillator.stop(audioContext.currentTime + duration);
}
```

**设计要点**:
- 使用指数衰减避免音效突兀结束
- 音量控制在0.3，避免过大
- 懒加载初始化音频上下文

#### 便捷音效函数

```javascript
// 播放"知道"音效
export function playKnowSound() {
    const config = AUDIO_CONFIG.KNOW;
    playAudio(config.frequency, config.duration, config.type);
}

// 播放"不熟"音效
export function playUnfamiliarSound() {
    const config = AUDIO_CONFIG.UNFAMILIAR;
    playAudio(config.frequency, config.duration, config.type);
}

// 播放显示释义音效
export function playShowMeaningSound() {
    const config = AUDIO_CONFIG.SHOW_MEANING;
    playAudio(config.frequency, config.duration, config.type);
}
```

### 2.2 Web Speech API 语音合成

#### 简单语音朗读（不带会话锁）

```javascript
export function speakWord(text, lang = 'en-US') {
    if (!('speechSynthesis' in window)) return;
    
    const rate = SPEED_CONFIG[state.currentSpeed]?.rate || 0.8;
    
    window.speechSynthesis.cancel();
    const utterance = new SpeechSynthesisUtterance(text);
    utterance.lang = lang;
    utterance.rate = rate;
    window.speechSynthesis.speak(utterance);
}
```

#### 会话锁安全语音朗读

```javascript
export async function speakWordAsync(text, lang = 'en-US') {
    if (!('speechSynthesis' in window)) return;
    
    const capturedSessionID = state.playSessionID;
    const rate = SPEED_CONFIG[state.currentSpeed]?.rate || 0.8;
    
    return new Promise((resolve) => {
        const utterance = new SpeechSynthesisUtterance(text);
        utterance.lang = lang;
        utterance.rate = rate;
        
        utterance.onend = () => {
            if (state.playSessionID !== capturedSessionID) {
                resolve({ status: 'aborted' });
                return;
            }
            resolve({ status: 'success' });
        };
        
        utterance.onerror = () => {
            resolve({ status: 'error' });
        };
        
        window.speechSynthesis.cancel();
        window.speechSynthesis.speak(utterance);
    });
}
```

**会话锁机制**:
- 在开始语音合成前捕获当前 `playSessionID`
- 在 `onend` 回调中检查会话ID是否仍然有效
- 如果会话已过期，返回 `{ status: 'aborted' }`
- 如果会话有效，返回 `{ status: 'success' }`

### 2.3 音频上下文管理

```javascript
let audioContext = null;

function initAudioContext() {
    if (!audioContext) {
        audioContext = new (window.AudioContext || window.webkitAudioContext)();
    }
    return audioContext;
}
```

**懒加载策略**:
- 首次播放音效时才初始化音频上下文
- 避免页面加载时不必要的资源消耗
- 兼容旧版浏览器（webkit前缀）

## 四、典型使用场景

### 场景1: 用户标记"知道"

```
1. 用户按 K 键
2. transition('KNOW') 被调用
3. playKnowSound() 播放880Hz正弦波音效
4. 用户获得即时反馈
```

### 场景2: 用户标记"不熟"

```
1. 用户按 J 键
2. transition('UNFAMILIAR') 被调用
3. playUnfamiliarSound() 播放220Hz锯齿波音效
4. showUnfamiliarIndicator() 显示"不熟"动画
5. 用户获得即时反馈
```

### 场景3: 单词朗读

```
1. displayWord(word) 被调用
2. speakWord(word.word) 朗读单词
3. 语音合成使用当前速度配置
4. 如果用户快速切换，会话锁中止旧的语音合成
```
